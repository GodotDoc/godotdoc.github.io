    TileSet #

    TileSet 是一个图块集，也就是能放置到 TileMap (或者TileMapLayer)上的图块的集合。

    创建 TileSet 可以借助 TileMap 的检查器窗口。

    如果想要重复使用，可以点击 TileSet 那行最右边的向下的小三角点击“保存”，下次别的TileMap使用的时候点击加载就行了。

    设置 TileSet 属性 #

    创建完 TileSet 有几个基本属性

    • Tile Shape ：每个图块单元的形状，默认的是 Square(正方形)。还有其他形状，比如：Isometric(等角)、Hexagon(五边型)等。
    • Tile Layout：图块的布局方式。
    • Tile Offset Axis：图块偏移轴，可以水平偏移或者垂直偏移。
    • Tile Size ： 每个图块的大小，这个一般跟 TileMap 的 cell quadrant size 属性一样。

    设置 TileSet 图块 #

    点击新建的 TileSet（1） 会在下面出现个设置图块的窗口（2）。将准备好的图块资源拖拽到图块窗口中（3）。
