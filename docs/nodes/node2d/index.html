    Node2d

    Node2d #

    2D 游戏对象，具有变换（位置、旋转、缩放）等基本功能。所有的 2D 节点，包括物理对象和精灵，都继承自 Node2D。可以使用 Node2D 作为父节点来移动、缩放和旋转 2D 项目中的子节点。
