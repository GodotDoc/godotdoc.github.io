    GDScript 语法参考 #

    GDScript 的语法很简单，下面通过一些案例帮助读者快速的入门 GDScript。

    本文默认读者至少有少许的编程经验，知道变量、常量、对象和类等词是什么意思。

    没有特殊说明，下文的代码都是在 func _ready(): 方法中编写的，然后点击右上角可以运行看效果。

    变量 #

    变量是用于存储信息的"容器"，这个"容器"储存的数据可以在游戏运行时随时更改，使用 var 关键定义。

    # 定义一个叫 n 的变量
var n = 1
# 使用冒号和数据类型关键字手动声明变量类型
var n1 : int = 1
# 可以省略类型关键字，系统会自动推断
var n2 : = 1

# 修改变量值
n = 2
n1 = 2
n2 = 2

    是否声明变量类型根据自己喜好来。笔者觉得声明变量类型使用起来比较方便，不声明类型代码看起来简洁。

    常量 #

    常量是在游戏运行时无法更改的值，使用关键字 const 定义。

    # 常量就是定义了就不能修改的数据"容器"
const A = 1

    注释 #

    注释是用来给程序员看的"代码"，不会被系统编译，相当于代码的备忘录，使用关键字 #。

    # 这里就是一条注释，可以理解为代码的备忘录，程序运行的时候，会把这行删掉。
var n = 1
const N = 1

    从 Godot4.2 开始，可以使用双井号 ## 注释。

    双井号注释被称为"文档注释"，可以像官方注释一样生成代码文档，也可以在检查器(Inspector)中鼠标悬停显示注释内容。

    extends Node2D

## 这是个变量的文档注释
@export var a = 1

## 这是个函数上的文档注释
func test_function():
	pass

    数据类型 #

    Godot 内置一些数据类型，分为基本数据类型(int、float、String、bool等)和对象数据类型(Dictionary、Array等)。基本数据类型在当作参数传递的时候是"按值传递"。对象数据类型在当作参数传递的时候是"引用传递"。

    数字类型 #

    整数类型 # 

    var n = 1
var n1 : int = 2

    int 存储的是 64 位值，等效于 C++ 中的 int64_t

    浮点数类型 # 

    var f = 1.1
var f1 :float = 1.2

    float 存储的是 64 位值，等效于 C++ 中的 double。

    字符类型 # 

    var s = "Hello Godot!"
var s1 : String = "Hello Godot!"

    布尔类型 # 

    var b = false
var b1 : bool = true

    默认使用 Unicode 字符。

    字典类型 # 

    var dict  =  {"key": "value", 2: 3}
var dict1 : Dictionary = {key = "value", other_key = 2}

# 返回该字典中键为 key 的值
var v1 = dict.get("key")
print(v1) # 输出 value

# 如果字典中没有名为 key1 的键，返回 null
var v2 = dict.get("key1")
print(v2) # 输出 null

# 不存在的时候返回某个默认值
var v3 = dict.get("key1","default value")
print(v3) # 输出 default value

# 合并两个字典
dict.merge(dict1) # 将 dict1 中的值合并到 dict 中
var v4 = dict.get("other_key")
print(v4) # 输出 2

# 移除键值
dict.erase("other_key")
var v5 = dict.get("other_key")
print(v5) # 输出 null

# 添加键值
dict["other_key"] = "new_value"
var v6 = dict.get("other_key")
print(v6) # 输出 new_value

# 更新某个键的值
dict["other_key"] = "updated_value"
var v7 = dict.get("other_key")
print(v7) # 输出 updated_value

    字典的值类型可以是任何类型。

    数组 # 

    var arr = [1, 2, 3]
var arr1 : Array = ["string1","string2","string3"]

# 添加值
arr.append(4)
print(arr) # 输出：[1, 2, 3, 4]

# 更新
arr[0] = -1 # 将索引为 0 位置的值改成 -1
print(arr) # 输出：[-1, 2, 3, 4]

# 取某个位置的值
var v1 = arr[1] 
print(v1) # 输出：2

# 移除某个值
arr.remove_at(0) # 移除索引为 0 位置的值
print(arr) # 输出：[2, 3, 4]
	
# 合并数组
arr.append_array(arr1)
print(arr) # 输出：[2, 3, 4, "string1", "string2", "string3"]

    数组设置类型 #

    上面的例子可以看出默认的 Array 类型是可以储存不同的数据类型，如果想只存某种类型可以使用“类型化数组”。

    # 只能存 int 类型
var a: Array[int]

# 只能存 Node 类型
var b: Array[Node]

# 只能存某种枚举类型
var d: Array[MyEnum]

    Array 等价于 Array[Varaint]

    压缩数组 #

    如果读者需要使用数组存储较多的值（数万个），建议使用压缩数组。压缩数组可以避免了内存碎片，从而降低内存的使用量。但是压缩数组比普通的数组性能要低一点，所以不建议在使用小数据集的时候使用压缩数据。

    var arr : PackedInt64Array = [1,2,3]
print(arr) # 输出：[1, 2, 3]

var arr1 : PackedStringArray = ["string1","string2","string3"]
print(arr1) # 输出：["string1", "string2", "string3"]
    • PackedByteArray：字节（从 0 到 255 的整数）数组。
    • PackedInt32Array：32位整数数组。
    • PackedInt64Array：64位整数数组。
    • PackedFloat32Array：32位浮点数数组。
    • PackedFloat64Array：64位浮点数数组。
    • PackedStringArray：字符串数组。
    • PackedVector2Array：Vector2 类型的数组。
    • PackedVector3Array：Vector3 类型的数组。
    • PackedColorArray：Color 类型的数组。

    运算 #

    数值运算 # 

    var n1 = 1
var n2 = 2

# 加
var n3 = n1 + n2
print(n3) # 输出 3

# 减
var n4 = n1 - n2
print(n4) # 输出 -1

# 乘
var n5 = n1 * n2
print(n5) # 输出 2

# 除
var n6 = 1.0 / 2.0
print(n6) # 输出 0.5
	
# 取余
var n7 = 10 % 3
print(n7) # 输出 1

    枚举 # 

    extends Node2D

# 匿名枚举类型
enum {UNIT_NEUTRAL, UNIT_ENEMY, UNIT_ALLY}

# 一个名称为 Name 的枚举
enum Name {THING_1, THING_2, ANOTHER_THING = -1}

func _ready():
	
	print(UNIT_NEUTRAL) # 输出：0
	
	print(Name.THING_1) # 输出：0
	
	print(Name.ANOTHER_THING) # 输出：-1

    枚举类型实际上存的的是就是数字类型，一些地方使用枚举类型可以提高性能和可维护性。
