    GDScript #

    Godot官方提供多种开发语言供开发者选择，如：GDScript、C# 、C/C++、其他。

    GDScript 是 Godot 专用的编程语言，具有设计简洁、学习曲线低、与Godot引擎紧密集成等优点，非常适合新手和独立开发者。

    如果你有其他面向对象语言开发基础，学习 GDScript 将会很快。
